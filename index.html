Mis trabajos

Una colección de mis proyectos más recientes.

Domposer

Domposer is a note-taking tool which has a UI that's unique, quickly add and arrange HTML and CSS, upload and download created projects and files, sign in to save up projects and can be installed as a Progressive Web Application. The stack is Laravel for the backend API's and vanilla JS for the frontend. The GitHub repository for this is private at the moment.

Bay.js

An easy to use lightweight library for web components. It doesn't need a build step but can be included in a build step if you want to. It's a great way to create reusable components for your projects. It's available as a NPM package and doesn't use any dependencies and is ~4kb minified. Below doesn't use even that much. Functions can be used in and CSS policies without a build step.

Cookiemunch

A simple, customizable minimal setup cookie plugin that allows your users to select which cookies to accept or decline. This was created with vanilla JS, SCSS and Parcel Bundler and is available as a NPM package and the git repository makes any type of custom parent cookie and themes possible.

Screen time converter

Count down the time you (or the spender) are allowed to browse social media, chores and other websites. It's a great way to keep track of how much time you spend on certain websites and can be used as a parental control tool.

Inline svg

This is a NPM package that takes an SVG file and renders it as inline SVG to the DOM. This is useful if you want to manipulate the SVG with CSS or JavaScript. It's also useful if you want to include SVG in your HTML without having to create a separate file.

PCDJ

We had the opportunity to work with PCDJ DJ software, where I contributed to the front-end development using Vue.js. This involved creating new features, improving existing ones, and ensuring a seamless user experience for their professional DJ software.
